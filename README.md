# MetaSearch Universal v3.0

## Méta-Moteur Android avec Multi-Agents IA

### Interface
- Material Design (KivyMD)
- Dark theme moderne
- Bottom navigation intuitive
- Cards et chips Material

### Fonctionnalités
- 🤖 Multi-Agents IA parallèles
- 🔄 Auto-update intégré
- 🌐 Proxy Manager (scraping + test)
- 🔐 Mode Master
- 🧅 Tor compatible

### Compilation
```bash
pip install buildozer cython
buildozer android debug
```

### Installation
1. Télécharger l'APK depuis GitHub Actions
2. Activer "Sources inconnues"
3. Installer
